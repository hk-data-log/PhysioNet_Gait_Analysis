# 🏃‍♂️ PhysioNet 보행 EMG 데이터 분석 
> **임상적 지식과 데이터 엔지니어링의 융합 (최종 업로드 완료 🚧)**

본 프로젝트는 **PhysioNet Gait in Elderly Adults** 데이터셋을 정복하기 위한 치열한 기록임. 단순히 예쁜 결과만 나열하는 것이 아니라, 대규모 생체 신호 데이터를 다루며 마주한 기술적 한계와 이를 극복해 나가는 ** 분석 과정**을 담았음.

---

### 🚀 주요 작업 내용 (Core Contributions)

* **데이터 정렬 및 무결성 확보:** `sort` 함수를 이용한 논리적 정렬 체계를 구축하여 31명 피실험자 데이터 간의 일관성을 확보함.
* **직관적 리팩토링:** 도메인 지식을 반영하여 `Subject_ID` -> `ID`, `LT_Peak`, `Asymmetry` 등으로 변수명을 최적화하고 코드 가독성을 높였음.
* **메모리 최적화:** `inplace=True` 파라미터를 적극 활용해 불필요한 복사본 생성을 방지하고, 대용량 데이터프레임을 효율적으로 제어함.
* **심층 시각화 및 인사이트:** **Box Plot**으로 데이터 분포와 이상치를 감별하고, **Heatmap**으로 근육 간 상관관계를 분석하여 시각적 설득력을 강화함.
* **대규모 데이터 핸들링:** 60만 줄 이상의 로우 데이터를 정제하며 발생한 경로 및 로딩 오류를 해결하고 안정적인 분석 파이프라인을 설계함.

### 💡 삽질의 기록: 코드와 함께 성장한 원칙들

매일 에러와 사투하며 정립한 본인만의 데이터 분석 원칙임.

1. **"에러는 발견 즉시 검거한다"**
   부채처럼 쌓인 에러는 결국 시스템 전체를 무너뜨림. '나중에'라는 미련을 버리고 현재 셀에서 문제를 완결짓는 것이 가장 빠른 지름길임을 배움.

2. **"변수명은 데이터의 신분증이다"**
   모호한 이름은 분석을 미궁으로 빠뜨림. 명확한 네이밍이 곧 코드의 품질이자 실력임을 깨달음.

3. **"어제의 나를 믿지 말고 주석을 믿자"**
   기록되지 않은 로직은 휘발됨. 미래의 나를 위해 설계 의도를 상세히 남기는 시스템적 사고의 중요성을 절감함.
